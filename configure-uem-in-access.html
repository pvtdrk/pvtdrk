Configure UEM in Access

pvtdrk

More paint by numbers for this overview of a very simple integration as always refer to VMware documentation for the latest guides and recommendations but this should help as a visual guide. In brief we need to create some API keys an Admin with a cert and punch the details into Access, lets jump in.

Navigate to the config page within Access

Head over to the UEM console to create API keys, the Admin and make a note of the Group ID

API Keys, you need an Admin and an Enrollment User key, Once generated copy them into the relevant field in the Access config page

The Group ID details 

We also need an Admin account with API Cert Auth enabled, once created we can go back in and export the certificate

Back to Access with all the info from the above steps and we can import the certificate and save the settings.

And this bit is complete.
